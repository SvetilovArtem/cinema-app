<h1>Cinema-app</h1>

<h3>Технологии: React, Redux Toolkit, axios, html/scss, bootstrap.

<h2>Описание</h2>
<p>В проекте предусмотрено 5 страницы: главная, лучшее, каталог, логин и избранные.</p>
